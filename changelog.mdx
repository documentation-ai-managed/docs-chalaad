<Update label="2024-09-20" description="v2.0.0" tags={["feature", "breaking"]}>

## New Features

- Redesigned dashboard with customizable widgets for your favorite devices.
- Added subscription service for recurring smart home accessory deliveries.
- Integrated voice control setup guides for Google Home and Alexa.

- Updated API endpoints from `/v1/products` to `/v2/products` for better performance.
- Removed legacy support for discontinued Zigbee 1.0 devices.

- Streamlined checkout process with one-click Apple Pay and Google Pay.
- Improved product ratings system with verified purchase badges.

## Bug Fixes

- Corrected pricing display errors for bundle discounts.
- Patched security vulnerability in webhook notifications.

</Update>
